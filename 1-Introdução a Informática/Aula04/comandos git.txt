Git

Comandosde inicialização e criação:

git init // cria um repositório

git config --global user.name nomeUsuario // agrega seu nome ao repositório

git config --global user.email nomeUsuario@email.com // agrega seu email ao repositório

git remote add origin https://github.com/DH/RepoRemoto // aponta um repositório remoto


Comandos add arquivos:

git add // adiciona todos os arquivos

git status // mostra o estado dos arquivos

git commit-m // adicionando o arquivo app.js (o commit adiciona data+msg+autor na alteração)



Comandos add multiplos arquivos:

git add . // Adiciona todos os arquivos ao repósitório
git status // lista todos os arquivos e alterações
git commit -m //adicionando todos os arquivos - o commit adiciona data+msg+autor na alteração 



git config --list

ls -R // lista de arquivos recursiva
